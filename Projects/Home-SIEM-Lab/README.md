# Home SIEM Lab using Splunk

## What I Built
Set up Splunk on my local computer to create a personal SIEM environment.
monitored and identified unusual activities using ingested Windows Event Logs.

## Tools Used
- Splunk (Free)
- Windows Event Logs

## What I Did
- Splunk was set up to gather login events from my computer.
- Constructed a dashboard to show unsuccessful login attempts.
- Developed an alert rule to identify brute-force trends(5+ failures/minute).

## Incident Detected
Detected multiple failed login attempts on 21/04. 
Traced source, documented findings, and resolved the alert.
